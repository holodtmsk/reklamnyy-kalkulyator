# СБТ ИИ-Технолог

Веб-приложение для автоматического расчёта смет рекламных конструкций на базе DeepSeek-R1 через Amvera LLM.

## Деплой на Amvera

### 1. Создай приложение в Amvera
- Зайди в «Приложения» → «Создать»
- Тип: **Docker** или **Python**
- Регион: Москва_0

### 2. Добавь переменную окружения
В настройках приложения → «Переменные окружения»:
```
AMVERA_TOKEN = <твой токен из раздела LLM → DeepSeek-R1>
```

### 3. Загрузи код
Через Amvera Online IDE или git:
```bash
git init
git add .
git commit -m "init"
git remote add amvera <url из amvera>
git push amvera main
```

Или загрузи ZIP через веб-интерфейс Amvera.

### 4. Настрой persistenceMount
В amvera.yaml уже прописан `/app/data` — там будет храниться SQLite база и прайс-лист.
Данные НЕ удалятся при перезапуске.

## Логины менеджеров
- sbt01, sbt02, sbt03, sbt04, sbt05
- Пароль не нужен — просто ввести логин

## Функции
- Чат с ИИ-технологом (DeepSeek-R1)
- История расчётов по каждому менеджеру
- Загрузка прайс-листа (PDF/TXT)
- Экспорт в Excel и TXT
- Полная логика расчёта ФОТ, материалов, ЧПУ согласно инструкции
